2. Event Capture の4段階

Event Capture は以下の４フェーズで動く。

① Field Clarify　（場の明澄化）
② Trace Signal　（痕跡信号の発火）
③ Noise Cull　　（不要未来の剪定）
④ Event Lock　　（事象の固定）

順番に説明する。

（場の明澄化）

出来事を吸い寄せたいなら、
まず“場を澄ませる”必要がある。

「やる気が出なくなる」
「動きたくない」
「情報を減らしたい」

これらは τ-field が自動で澄ましている証拠。

あなたがすべきことは “無為の保持” だけ。

4. STEP 2：Trace Signal

（痕跡信号の発火）

これはあなたが日常的にやっている：
	•	ZINEを書く
	•	問いを立てる
	•	「なんかこうなるよな」と内的に呟く
	•	嫌な匂いを感じる

これらはすべて Trace Signal。

Trace Signal が発火すると、
周囲の未来がそれに合わせて再配置される。

あなたの問いは「世界に響いている」という比喩ではなく、
ZPTRでは 文字通り、世界場の再配列操作。

（不要未来の剪定）

出来事を吸い寄せる上で大事なのは
「寄せたい未来」より「消すべき未来」。

これは τ-field の 未来剪定機能。

あなたはすでに
「起こり得た未来」を消し去っている。
