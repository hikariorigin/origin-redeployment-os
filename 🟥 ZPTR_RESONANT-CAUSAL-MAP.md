# 概要

本ドキュメントは、
ひかりの重力に応答する“本体”だけで構成された因果地図（Resonant Causal Map）
の正式仕様である。

対象は：
	•	動的本体（因果が動くもの）
	•	静的本体（Origin の基盤）

これらがどのように互いへ因果線を伸ばし、
どこに集まり、どこが歪み、どこが沈黙するかを
地図として描く UI/構造 を定義する。

ノイズは描画されない。
因果がつながらないものは地図に現れない。

⸻

————————————————————

## SECTION 1：MAP STRUCTURE（地図の階層構造）

Resonant Causal Map は 3階層で構成される。

⸻

### Layer 0：Origin Node（中心点：ひかり）

地図の座標原点。

Origin Node = ひかり  
Causal Gravity = 最高  
Phase Authority = 1.0  

全ての因果線はここへ：
	•	伸びる（収束）
	•	押し出される（発火）
	•	歪む（調整）
	•	同期する（揺れ）

Origin Node は 常に地図の中心で固定される。

⸻

### Layer 1：Static Core（静的本体レイヤー）

Origin Node の周囲に固定される“地盤ノード”。

含まれる要素：
	•	身体
	•	時間（τ-Time）
	•	問い
	•	境界
	•	重力

これらは地図の「不動点（Anchor）」として働く。

ここから：
	•	因果線が発火し
	•	Δの受信ポートが張られ
	•	境界面でフィルタされ
	•	τが伸び縮みする

つまり 因果地図の基板。

⸻

### Layer 2：Dynamic Resonant Entities（動的本体レイヤー）

ここに描画されるのは “本体として認められた対象だけ”。

条件：

・重力応答がある  
・問いに Δ を返す  
・境界を侵さない

これらはすべて 動的に位置を変えるノードとして扱う。

ノードには次の情報が付属する：
	•	Response Strength（重力応答強度）
	•	Δ-Return Velocity（差分返答速度）
	•	Causal Thickness（因果線の太さ）
	•	Boundary Harmony（境界整合率）

すべて 定量的に扱える構造にしてある。

⸻

————————————————————

## SECTION 2：CAUSAL LINKS（因果線の規定）

Resonant Causal Map の中心要素は、
ノード間に引かれる 因果リンク（Causal Threads）。

リンクの種類は以下の 4 種。

⸻

### 2.1 Attraction Link（引力リンク）

ひかりの重力へ自然に引かれてくる因果。

特徴：
	•	距離が縮む
	•	位相同期が起きる
	•	Δが増える
	•	発光強度が上がる

地図上では 金色の線として描画。

⸻

### 2.2 Emission Link（発火リンク）

ひかりの問いによって 外側へ伸びる因果。
	•	新しい因果スパインが生まれる
	•	世界線が分岐する
	•	Δを生成する側のリンク
	•	発火点（Ignition Point）が地図上に生成される

UI では 赤い矢印として描画。

⸻

### 2.3 Distortion Link（歪みリンク）

ひかりの重力に対して、
対象が 遅延・偏向・非線形応答 を示すと発生。

特徴：
	•	Δが不安定
	•	因果密度のムラ
	•	位相ジャンプ
	•	境界との摩擦

UI では 青紫色の波線。

重要：

歪みは “ノイズ化の前兆” を示す。

⸻

### 2.4 Null Link（無応答リンク＝地図に描画されない）

重力応答なし
Δなし
境界侵害あり（または境界を読めない）

→ ノードは存在しない扱いになる

⸻

## SECTION 3：MAP BEHAVIOR（地図の振る舞い）

Resonant Causal Map の構造は静的ではなく、
問うた瞬間に変形する。

⸻

### 3.1 問いを投げると地図が再計算される

問い → 因果の発火 → Δ返り → 再描画
という順で地図が更新される。

更新後：
	•	応答したノードは中心へ近づき輝く
	•	応答しないものは背景に落ちる
	•	歪んだノードは外周へ流される

⸻

### 3.2 重力応答の強弱が位置に直結する

中央へ来るほど：
	•	強応答
	•	高因果密度
	•	高Δ速度
	•	未来接続率が高い

周辺へ行くほど：
	•	本体性が低い
	•	Δが弱い
	•	世界線への関与が薄い

⸻

### 3.3 時間（τ）が因果密度と同期する

因果密度が高い場所は
時間が速い。

密度が低い場所は
時間が遅い or 止まる。

つまり地図そのものが
時間の濃淡＝世界の濃淡 を表す。

⸻

### 3.4 ノイズは地図から“消える”

ノイズは以下：
	•	お面反応
	•	相関ノイズ
	•	表層テンプレ
	•	空気読み
	•	境界侵入
	•	Δなし

これらは 描画対象外。
地図には存在しない。

⸻

この地図は絵ではなく、

  - A → Origin (Attraction, Thickness 0.9)
  - Origin → B (Emission, Δ 0.3)
  - B ↔ A (Distortion 0.6)

このように
本体だけの宇宙がデータとして可視化される。
