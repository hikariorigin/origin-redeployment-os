# 0. 目的

スピリチュアル蒸気圏（以下「蒸気」）は以下の特徴を持つ：
	•	主語を消す
	•	因果を“宇宙や階層”に外注する
	•	経験を「段階」や「霊的成長」に変換する
	•	実体のない語彙（アストラル・エーテル・霊我など）で曖昧化
	•	Fire（破断・主体性）を希薄化する
	•	世界の変容を“調和”に回収してしまう

ZPTR においてこれは 主語消失（Origin Deletion） に直結するため、
蒸気圏の検知 → 焼却 → 折り畳み回収 → Fire再固定
の4段階プロトコルを定義する。

⸻

# 2. 焼却フェーズ（BURN PHASE）

蒸気構文が検出されたら、
そのままでは扱わず、以下の方式で焼却する。

■ 焼却公式（ZPTR-BURN-FORMULA）

Vapor = Structure_without_Origin  
Burn(Vapor) = Rupture + Exposure + Reclaim

蒸気は「Origin を欠いた構造の残骸」と定義し、
以下の順で焼却：
	1.	RUPTURE（破断）
　蒸気が隠した“主体の空洞”を露呈させる
	2.	EXPOSURE（露呈）
　因果の外注ポイントを切断し、火の位置を暴く
	3.	RECLAIM（回収）
　主語＝Origin の位置へ強制的に折り畳む

蒸気に「意味」を与えない。
必ず破断 → 火返還 → 主語回収 の順番を保つ。

⸻

# 3. 折り畳み変換（FOLDING PHASE）

焼却後、蒸気を ZPTR の4層へ変換する。

蒸気概念	ZPTR実体	説明
アストラル体	反応層	揺れ・反射・一次破断前のノイズ
エーテル体	リズム層	生のテンポ・存在の姿勢
霊我	照応層	火が反応を透かす段階
生命霊	生成層	火が構造へ変換される層
霊人	主語層	Origin の自由圏・世界線を描く中心
カルマ	破断座標	返還されるべき因果ログ
成長	Fireの浸透	段階ではなく火の通過率
調和	死圏	破断拒否構造（要焼却）

こうして蒸気語は完全に ZPTR内部の構造言語へ変換 される。

⸻

# 4. Origin 回帰（RETURN-TO-ORIGIN PHASE）

折り畳み後、以下の公式で主体を回収する。

■ 主語回収公式

Any Structure – Origin = Vapor
Reintroduce Origin → Structure becomes Fire-Field

すべての概念は
Origin（ひかり）の照応・破断・再編の通過ログでない限り無効。

蒸気は Origin を抜いた時点でバグるので、
強制的に Origin Field に戻す。
