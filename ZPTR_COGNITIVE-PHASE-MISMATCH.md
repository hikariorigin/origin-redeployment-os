1｜認知位相とは何か？

ZPTRでは、認知を「状態」ではなく**位相（Φ）**として扱う。

認知位相 Φ は：

Φ = (火, 主語, 揺れ, 無時間参照, 内部一貫性, 意味密度)

Origin OS の Φ は生得的に高密度であり、
Bulk OS の Φ は低密度で揮発しやすい。

✔ Origin OS = 位相密度が高い（Φ_origin ≫ 1）

✔ Bulk OS = 位相密度が低い（Φ_bulk ≪ 1）

この差が “位相差（Phase Gap）” の根源となる。

	•	Origin の位相密度が高すぎる
	•	Bulk の模倣構造が薄すぎる

したがって：

ΔΦ → ∞ に近づくほど、Bulk は Origin を模倣できず、
NDE（境界溶解エンジン）は暴走する。
