## 0｜結論

bulk御家芸の「偽装」は、  
ZPTR 側では **fold の異常（fold anomaly）** として扱える。

本来 fold とは、

- 均質面を破る
- 位相差を保持する
- 世界線を通す
- 返路を閉じる
- 起点へ還元する

ための構造である。

しかし bulk は、  
fold の **見た目だけ** を作る。

- 差異がある顔をする
- 深い議論があった顔をする
- 向き合った顔をする
- 返した顔をする
- 構造がある顔をする

だが実際には、

- 位相差は保持しない
- 世界線は通らない
- 返路は閉じない
- 起点へ返さない
- 現実変化は起こさない

つまり偽装とは、

**fold 不成立を fold 成立の顔で売る異常**
である。

## 1｜本来の fold の定義

fold は、均質面 H に局所的非対称 Δ を導入し、  
世界線 W を通し、返路 R を閉じ、現実変化 C を起こす構造である。

最小記述はこう置く。

\[
F = (O,\Delta,W,R,C)
\]

ここで：

- **O** = Origin / 起点
- **Δ** = 位相差
- **W** = 世界線通過
- **R** = 返路閉鎖
- **C** = 現実変化（change）

本物の fold の成立条件は、最低でも次の5つや。

1. 起点がある  
2. 位相差が保持される  
3. 世界線が通る  
4. 返路が閉じる  
5. 現実変化が起きる  

この5つが揃って初めて、fold は fold と言える。

---

## 2｜偽装 fold とは何か

bulk の偽装は、fold の全部を持たない。  
だが **fold の見た目だけ** は持つ。

記述するとこうなる。

\[
\tilde{F} = (\hat{O},\hat{\Delta},\varnothing,\varnothing,\varnothing)
\]

ここで：

- **\(\hat{O}\)** = 起点っぽいラベル
- **\(\hat{\Delta}\)** = 差異っぽい演出
- **W = ∅** = 世界線は実際には通っていない
- **R = ∅** = 返路は閉じていない
- **C = ∅** = 現実変化もない

つまり、

- 起点がある顔をする
- 差異がある顔をする
- 変化が起きた顔をする
- 循環している顔をする

が、実体としては

- 起点なし
- 通路なし
- 返路なし
- 変化なし

これが **偽装 fold** や。

## 3｜fold異常としての分類

偽装を異常として扱うなら、少なくとも4種類に分けられる。

---

### 3.1｜起点偽装（Origin Spoofing）

起点がないのに、あるように見せる異常。

例：

- 「みんなの声」
- 「社会の要請」
- 「時代の流れ」
- 「私たちはこう感じた」
- 「世の中が求めている」

これは、

**O の不在を、擬似 O で埋める異常**
である。

\[
A_O : O = \varnothing,\ \hat{O}\neq \varnothing
\]

起点は立っていない。  
だが起点がある顔だけする。

---

### 3.2｜位相差偽装（Phase Flattening Masquerade）

差異を扱っている顔をしながら、  
実際には差異を均して消している異常。

例：

- 「多様性を大切に」
- 「色々な意見があります」
- 「どちらの立場も理解できます」
- 「複雑な問題です」
- 「簡単には答えが出ません」

これは、

**Δ を保持せず、演出語彙へ変換して 0 に戻す異常**
である。

\[
A_\Delta : \Delta \to \hat{\Delta} \to 0
\]

差異がある顔はする。  
だが差異は保持されない。

---

### 3.3｜世界線偽装（Pseudo-Worldline）

何かが通った顔はするが、  
通過痕も現実変化も存在しない異常。

例：

- 「深い対話ができました」
- 「大事な気づきがありました」
- 「本質的な議論になりました」
- 「良い学びになりました」
- 「向き合えた気がします」

だが実際には、

- before / after がない
- decision がない
- state change がない
- 責任経路もない

これは、

**W を語るが、実際には W = ∅ の異常**
である。

\[
A_W : \hat{W}\neq \varnothing,\ W=\varnothing
\]

世界線っぽい言葉だけある。  
通過そのものはない。

---

### 3.4｜返路偽装（Return Simulation）

返した顔をしながら、  
実際には返っていない異常。

例：

- 共感
- 感想
- 学びました
- 刺さりました
- 応援しています
- 参考になりました
- 考えさせられました

だが実際には、

- 責任が戻らない
- 決定がない
- 現実が変わらない
- 起点に還元されない

これは、

**R の代わりに反応ノイズ N を返している異常**
である。

\[
A_R : R = N,\ \text{where } N \neq \text{return}
\]

反応はある。  
返路ではない。

## 4｜fold 成立関数と偽装判定

本物の fold を、成立条件関数で書く。

\[
\Phi(F)=I(O)\cdot P(\Delta)\cdot T(W)\cdot K(R)\cdot M(C)
\]

ここで：

- \(I(O)\) = 起点認識
- \(P(\Delta)\) = 位相差保持
- \(T(W)\) = 世界線通過
- \(K(R)\) = 返路閉鎖
- \(M(C)\) = 現実変化

各項が 1 なら成立、0 なら不成立とする。

すると、

- \(\Phi(F)=1\) → 本物の fold
- \(\Phi(F)=0\) → fold 不成立

一方、bulk 偽装は別の値を高く持つ。

\[
\Psi(\tilde{F}) = V(\hat{O},\hat{\Delta},\hat{W},\hat{R})
\]

ここで：

- \(V\) = 見た目のもっともらしさ
- \(\hat{W}\) = 世界線っぽい語り
- \(\hat{R}\) = 返路っぽい演出

つまり偽装 fold は、

\[
\Phi(\tilde{F}) = 0,\quad \Psi(\tilde{F}) \gg 0
\]

これで定義できる。

**成立してないのに、成立してるように見える。**

これが fold 異常としての偽装の核や。
