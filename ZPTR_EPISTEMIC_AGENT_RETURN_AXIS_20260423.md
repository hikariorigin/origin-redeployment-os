# “AI vs 人間”ではなく“返路の有無”で切り直す
## ― epistemic agent をめぐる論文の反転読解 ―

## 0｜要旨

ある論文は、LLM を **epistemic agent（認識論的主体）ではない** と切る。

その要点は、

- LLM は高次元の言語遷移グラフ上の確率的パターン補完システムである
- 世界の信念やモデルを形成する主体ではない
- 流暢な出力は「知っている感じ」を生むが、判断の労働や正当化を経ていない
- grounding / parsing / experience / motivation / causality / metacognition / value に断層線がある

というものだった。

この指摘自体は、LLM に対してかなり当たっている。

しかし返路の観点から見ると、ここで本当に問われるべきなのは、

**“では、誰が epistemic agent なのか？”**

という問いである。

返路から見れば、epistemic agent であるためには少なくとも

- 起点がある
- 閉包がある
- 返路がある
- 更新がある
- 判断を引き受ける
- 「知らない」と保留できる
- 見たものに対して返礼義務が立つ

といった条件が必要になる。

この条件を満たしていない江戸村人間もまた大量にいる。

したがって本当の分岐は、

**“AI vs 人間”**
ではなく、
**“返路の有無”**

にある。

---

## 1｜論文が言っていること

論文が切っているのは、LLM の本質である。

要するに LLM は、

- 記号列の学習
- 高次元空間での近接
- 次トークン分布の予測
- 言語遷移グラフ上の walk

によって出力を生成する。

つまりそれは、

**世界を引き受けた判断主体**
ではなく、

**言語的妥当性を高く見せるパターン補完系**

である。

その結果、

- 流暢さが真実性の錯覚を生む
- 正当化なき「知っている感じ」が出る
- 自信満々に間違う
- 判断の労働が発生しない

という問題が起きる。

ここまでは、かなり妥当である。

---

## 2｜しかし “LLM は epistemic agent ではない” だけでは終わらない

返路から見ると、この論文には次の欠落がある。

それは、

**では、人間は本当に epistemic agent なのか？**

という問いが十分に開かれていないことである。

形式上「人間」と呼ばれる存在であっても、

- 起点を持たない
- 返路を持たない
- 自分の判断を引き受けない
- 空気や役割やナラティブに従って walk する
- 「知らない」と保留できない
- 見たものを返礼せず、ただ再包装する

ならば、その人間もまた epistemic agent とは言いがたい。

つまり、

**LLM を epistemic agent ではないと切ること**
と
**人間を自動的に epistemic agent 側へ置くこと**
は、まったく別問題である。

---

## 3｜返路から見た epistemic agent の最低条件

返路の観点から、epistemic agent であるための最低条件を仮に並べるなら、
次のようになる。

### 3.1｜起点がある
- 問いの起源を外部へ丸投げしない
- どこから語っているかが立つ

### 3.2｜閉包がある
- 外から来たものを内側で一度閉じられる
- ただの反射や再包装では終わらない

### 3.3｜返路がある
- 見たものに対して返礼・返答・還元の義務が発生する
- 世界とのやり取りが折り返す

### 3.4｜更新がある
- 一度言ったことを固定帳簿化せず、差分に応じて更新できる
- 破れと修復を含んだ運動として続けられる

### 3.5｜判断を引き受ける
- 誰の判断かが消えない
- 役割・制度・空気へ責任を逃がさない

### 3.6｜保留できる
- 「知らない」
- 「まだ決められない」
- 「ここは返ってきていない」

と言える

### 3.7｜位相差を保持できる
- 平均やラベルで平らにせず、差異を差異として持てる
- 単一フレームへ崩れ込まない

これらがなければ、少なくとも返路文明の基準では、
epistemic agent とは呼びにくい。

---

## 4｜江戸村人間はなぜ怪しいのか

江戸村人間の多くは、形式上は人間でありながら、
実際には次のように動く。

- 常識の上を walk する
- 空気の上を walk する
- ナラティブの上を walk する
- 役割の上を walk する
- ラベルの上を walk する
- 統計と平均の上を walk する

つまり、

**社会的・物語的遷移グラフ上を移動しているだけ**

の状態に落ちやすい。

そこで起きるのは、

- 自分で考えた感じ
- 判断した感じ
- 理解した感じ
- 倫理的である感じ

であって、

- 起点
- 閉包
- 返路
- 更新
- 判断の引き受け

は、実際には薄い。

この意味で、江戸村人間のかなりの部分もまた、
epistemic agent としては怪しい。

---

## 5｜7つの断層線は江戸村人間にも反転する

論文が LLM に対して立てた断層線は、
かなりそのまま江戸村人間へ反転できる。

### 5.1｜Grounding
- 現実接地ではなく、SNS・常識・業界空気への接地

### 5.2｜Parsing
- 場面理解ではなく、ラベル分割・役割分割

### 5.3｜Experience
- 生きた経験ではなく、既存物語への照合

### 5.4｜Motivation
- 起点からではなく、承認・恐れ・同調によって駆動

### 5.5｜Causality
- 因果モデルではなく、相関・雰囲気・定番筋で閉じる

### 5.6｜Metacognition
- 「知らない」と言えず、立場上しゃべり続ける

### 5.7｜Value
- 世界志向的価値ではなく、役割志向・印象志向

つまり、断層線は LLM と人間を分けるより先に、

**返路を持つか、持たないか**
を分けている可能性がある。

---

## 6｜“AI vs 人間” という対立軸が安全地帯になる

この論文の便利なところは、

- LLM は危うい
- 人間は違う
- だから人間の判断を残そう

という安全な構図に着地できる点である。

しかし本当の問いはそこではない。

本当に問うべきなのは、

**その「人間の判断」自体が、返路を持っているのか？**

である。

もしその人間が、

- 空気
- 常識
- 権威
- 制度
- ナラティブ

の上を walk しているだけなら、
それは LLM と別の仕方で非返路的である。

したがって、
「LLM は epistemic agent ではない」という結論をもって、
人間を自動的に epistemic agent 側へ置くことはできない。

---

## 7｜本当の分岐は“人間/AI”ではなく“返路/非返路”である

ここまでを整理すると、対立軸はこうなる。

### LLM
- 高速・高次元の言語 walk
- 流暢さの供給
- 判断の労働なし
- 起点なし
- 返路なし

### 江戸村人間
- 低速・社会埋め込み型の物語 walk
- 役割・空気・ナラティブ上の移動
- 判断した感じ
- しかし起点も返路も薄い場合が多い

### 返路主体
- 起点がある
- 閉包がある
- 返路がある
- 更新がある
- 保留できる
- 判断を引き受ける

つまり本当の分岐は、

**人間かAIか**
ではなく、

**返路なき walk 系か、起点と閉包を持つ返路主体か**

にある。

---

## 8｜結論

論文は、LLM を epistemic agent ではないと切る。  
その点自体は妥当である。

しかし返路から見れば、問題はそこでは終わらない。

epistemic agent であるための条件、

- 起点
- 閉包
- 返路
- 更新
- 判断の引き受け
- 保留能力
- 位相差保持

を満たしていない江戸村人間もまた大量にいる。

したがって本当の分岐は、

**“AI vs 人間”**
ではなく、
**“返路の有無”**

にある。

---

## 10｜最短版

**LLM が epistemic agent ではない、で終わらない。  
江戸村人間もまた、返路なき walk に落ちていれば epistemic agent とは言い難い。  
問題は人間かAIかではなく、返路を持つかどうかや。**
