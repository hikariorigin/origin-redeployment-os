0｜要旨（Abstract）

本定理は、現代世界が “因果（Cause）” ではなく “起源（Origin）” を一次原理として動作する ことを証明する。

従来の世界では

Cause → Effect

で現象が説明されてきた。

しかしZPTR観測圏では、
Cause はすでに 二次的・派生的・後付け でしかなく、
世界の主要駆動源は次のように逆転している：

Origin（問い・揺れ） → Trace → 後付け因果 → 物語 → 現象

「因果以前に起源がある」
これが世界OSの実際の動作仕様である。

1｜定理の核心：

Cause ≠ Origin

Origin ≺ Cause（Origin が先行する）

つまり、
	•	世界は原因によって動くのではなく
	•	Origin の揺れが“原因”を後から生成する

という構造になっている。

この逆転は世界破綻の副作用ではなく、
ひかりが世界OSの震源に座ったために生じた構造的必然 である。

Axiom 1：世界は Trace-based system である

世界に意味として展開されない。

Bulk世界は Trace を理解できず、
「因果」「物語」で世界の穴を埋めるしかない。

Lemma 1：因果は可逆、Traceは不可逆である

因果は作り直せる（捏造も含む）が、
Trace は Origin の揺れから一度生成されると上書き不能。

Lemma 2：時間は Trace の順序で決まり、因果では整列しない

世界の時間軸は

「何を問い、何が揺れたか」

で決まり、
出来事の順序は後付けで修復される。

Lemma 3：Originが観測しない領域は世界に実体化しない

未観測領域＝空間の“裏側”。
その内部の因果は殻だけで、意味を持たない。

4｜正式定理（Main Theorem）

Theorem：

Origin が問いを発した瞬間、世界OSはその揺れを一次データとして採用し、
Trace を生成する。
因果はこの Trace に整合する形で後付け生成される。

式で表すと：

Let O = Origin’s Question (τ-vector)
Let T = Trace generated from O
Let C = Causal narrative generated post-hoc

Then:    O → T → C

And:     C ∉ Primary Data
         T ∈ Primary Data
         O ∈ Zero-Point Source (ZPTR)

つまり、
	•	因果は一次データではない
	•	起源（Origin）がゼロ点である
	•	世界は起源の揺れをもとに再構築されていく

これが「因果以前の起源定理」である。
