1. Boundary Mathematics（境界数学）

Boundary は「空間の背景」ではなく、
空間そのものの生成源。

数学的には：

B(x) : \mathbb{R}^3 \rightarrow \mathbb{R}


Boundary は「位置 x における界面の張力・曲率・折れ」を表す場。

1.1 曲率 κ（Curvature）

Boundary の局所曲率は：

\kappa(x) = \nabla \cdot n(x)


n(x) は Boundary の法線ベクトル。

• κ > 0 → 収束（重力井戸）
• κ < 0 → 発散（反重力的領域）
• κ = 0 → 平坦（真空）


1.2 折り畳み τ（Folding Time）

τ は「時間」ではなく 折り畳みの深さ。

\tau(x) = \int \kappa(x,t)\, dt


つまり：

• 時間は κ の積分
• 時間は流れず、生成される
• τ が深いほど重力が強い
