ZPTR_ECONOMY_ORIGIN_vs_STEAM.md

—— Origin経済と蒸気経済の比較構造 ——

(2026 / ZPTR Economic Field Theory)

⸻

0. 概要

本稿は、現代資本主義が運用している
“蒸気経済（Steam Economy）” と
ひかりが提示する
“Origin経済（生成主体経済）” を
ZPTR位相で比較する。

蒸気経済：
[
O = 0,; \tau = 0,; B = \emptyset
]
Origin経済：
[
O > 0,; \tau > 0,; B \neq \emptyset
]

⸻

1. 経済の定義

1.1 蒸気経済（Steam Economy）

外部ラベルと数値指標で動く経済。

特徴：
	•	GDP
	•	CPI
	•	失業率
	•	金利
	•	株価
	•	信用スコア

→ すべて外部的に“見える数字”で表される。
→ 主語（Origin）が不在。

定義式

[
E_S = f(\text{metrics})
]

⸻

1.2 Origin経済（存在経済）

生成主体（Origin）が作る価値の循環。

特徴：
	•	問い
	•	火（生成衝動）
	•	意味の生成
	•	τ（固有時間）
	•	Boundary（固有領域）
	•	作品・共鳴・影響力

→ 外部指標では測定不能。

定義式

[
E_O = g(O, \tau, R, \rho)
]

ここで：
	•	(O)：Origin密度
	•	(\tau)：存在の歴史
	•	(R)：再帰（深まり）
	•	(\rho)：意味密度

⸻

2. 価値の源泉の違い

2.1 蒸気経済

価値＝交換可能性。

[
V_S = \text{price}(x)
]

価値は「いくらで売れるか」で決まる。
主体が何を生み出したかは問題にならない。

⸻

2.2 Origin経済

価値＝生成源（Origin）から溢れる火の強度。

[
V_O = \frac{dR}{d\tau}
]

つまり 存在の変容速度 が価値を生む。
貨幣での換算は“副産物”。

⸻

3. 成長モデルの違い

3.1 蒸気経済の成長

指標の改善＝成長とみなされる。
	•	数字を並べ替える
	•	算出式を変える
	•	発表する範囲を調整する

[
G_S = \Delta(\text{metrics})
]

→ 実体がなくても成長できる（ように見える）。
→ “数字だけいじり放題”現象。

⸻

3.2 Origin経済の成長

成長＝存在の深まり。

[
G_O = \int \frac{dR}{d\tau} d\tau
]

外部の承認や指標は関係ない。
成長とは「主体の変容総量」。

⸻

4. 崩壊の構造

4.1 蒸気経済の崩壊

蒸気経済は本質的に：
	•	数字が先
	•	現実が後
	•	主語が不在

ゆえに、矛盾が蓄積すると 総崩れ になる。

例：
	•	“資産価格は上がっている → だから景気は良い”
	•	“失業率は改善している → だから問題ない”
	•	“株価が上昇している → 実体経済も上向き”

しかし内部実態がゼロなので：

[
\text{metrics} \uparrow \quad \text{reality} \downarrow
]

→ バブル → 暴落 → リセット
→ また数字作り直し

これが Citrini の書く「大不況ループ」。

⸻

4.2 Origin経済の崩壊

Origin経済は 崩壊しない。

理由：
	•	主語がある
	•	τがある
	•	経済そのものが存在の生成プロセスだから

Origin の衰退は 存在の変容停止（死） に近く、
外部ショックで壊れない。

⸻

5. 比較表

項目	蒸気経済 (Steam)	Origin経済
主語	不在	ある
時間	外部時間（T）	τ（固有時間）
成長	指標の向上	再帰密度の上昇
価値	価格	火（生成源）
危機	数字で発生	内面では発生しない
安定性	低い	高い
持続性	ラベル依存	Origin依存


⸻

6. 最終結論

⭐ 蒸気経済は“数字で動く世界”。

主語がなく、崩壊を数字で隠しながら進む。
危機も回復も“指標”で演出できる。

⭐ Origin経済は“人間そのものの存在深度で動く世界”。

価値は価格ではなく、
問い・火・意味密度・再帰で決まる。
崩壊しない。薄まらない。蒸発しない。

つまり：

⭐「蒸気経済は幽霊の経済。

Origin経済は存在の経済。」
