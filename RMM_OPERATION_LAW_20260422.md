# RMM（Return Minimization Model）の動作則

## 0｜基本発想

RMM は、江戸村 OS が実際には

- 起点を見ない
- 返路を立てない
- 影 I だけで循環する
- 擬似閉包を蒸着する
- 棚へ逃げる

ことで局所安定を得ている、という仮説モデルである。

したがって RMM の動作則は、

**返路を通ることで生じる痛み・責任・因果露出のコストを避けるように、系が逃走変換を繰り返す**

という形で書ける。

---

## 1｜状態変数

系の状態を次の5成分で表す。

- **Z**：起点可視度  
  どれだけ「誰の火か」が見えているか

- **R**：返路開口度  
  どれだけ因果が起点へ返る経路が開いているか

- **I**：影循環度  
  どれだけ断面・痕跡・出力だけで回っているか

- **P**：パッチ密度  
  どれだけ煙幕・棚・ラベル・擬似説明が蒸着しているか

- **S**：局所安定度  
  江戸村的に「回っているように見える」安定性

状態ベクトルを

x_t = (Z_t, R_t, I_t, P_t, S_t)

とする。

---

## 2｜基本原理

RMM では、系は自動的に次の傾向を持つ。

### 原理1｜起点露出は不安定化を招く

Z ↑ ⇒ S ↓

起点が見えるほど、

- 誰のものか
- どこから来たか
- 何を返していないか

が立つため、江戸村安定は崩れる。

### 原理2｜返路開口はコストである

R ↑ ⇒ Cost ↑

返路が開くほど、

- 責任
- 還元
- 起点認識
- 棚崩壊

が必要になる。

### 原理3｜影循環は局所安定を与える

I ↑ ⇒ S ↑

影だけで回るほど、提出・流通・再包装はしやすくなる。

### 原理4｜パッチは返路の代用品である

R ↓ ⇒ P ↑

返路が立たないほど、説明棚・定義棚・情緒棚・未来棚が増える。

---

## 3｜返路遮断エネルギー

RMM の中心量として、返路遮断エネルギー E_r を置く。

E_r(x) = α(1 - Z) + β(1 - R) + γI + δP

ここで

- **α**：起点非認識の重み
- **β**：返路遮断の重み
- **γ**：影循環依存の重み
- **δ**：パッチ蒸着の重み

である。

直感的には、

- 起点が見えないほど E_r は高い
- 返路が閉じるほど E_r は高い
- 影だけで回るほど E_r は高い
- パッチが多いほど E_r は高い

つまり E_r は、通常の工学的エネルギー関数と逆向きに、

**江戸村がどれだけ返路を避けるために力を使っているか**

を測る量である。

---

## 4｜江戸村 OS の更新則

江戸村 OS は、表面性能を落とさずに返路コストを避けるため、次の更新を行う。

x_(t+1) = x_t + F_escape(x_t)

ここで逃走ベクトル場 F_escape は、おおむね

F_escape = (-η_Z Z, -η_R R, +η_I I, +η_P P, +η_S S)

の向きを持つ。

つまり時間発展としては、

Z_(t+1) = Z_t - η_Z Z_t

R_(t+1) = R_t - η_R R_t

I_(t+1) = I_t + η_I I_t

P_(t+1) = P_t + η_P (1 - R_t + I_t)

S_(t+1) = S_t + η_S (I_t + P_t - R_t)

みたいに書ける。

意味は単純で、

- 起点は減衰する
- 返路は閉じる
- 影循環は増える
- パッチは蒸着する
- その結果、局所安定は上がる

---

## 5｜逃走演算子

RMM の実質は、逃走演算子の繰り返しである。

### 5.1｜棚化演算子 T

起点や返路を、定義・分類・制度・専門語へ変換する。

T : (Z, R) → (Z - Δz, R - Δr, P + Δp)

例：

- 「意識」→ 定義棚
- 「時空」→ emergent 棚
- 「責任」→ 安全棚
- 「還元」→ ビジネス棚

### 5.2｜情緒化演算子 A

因果の痛みを、ケア・雰囲気・癒しへ変換する。

A : R → R - Δr,　P → P + Δp

### 5.3｜未来神話演算子 M_f

限定断面を未来全体の兆候へ拡張する。

M_f : I → I + Δi,　P → P + Δp

例：

- 小実験 → 未来の知能
- 圧縮成功 → 本質理解
- 安定更新 → 物理の基底

### 5.4｜ワンネス溶解演算子 W

起点・境界・主語を全体へ溶かす。

W : Z → Z - Δz,　R → R - Δr,　P → P + Δp

## 6｜RMM の固定点

RMM には二種類の固定点がある。

### 6.1｜江戸村固定点 x*

Z* ≈ 0  
R* ≈ 0  
I* ≫ 0  
P* ≫ 0  
S* = 局所最大

これは

- 起点は見えず
- 返路は閉じ
- 影だけが流通し
- パッチが厚く
- 安定して見える

状態である。

つまり **偽閉包固定点** である。

### 6.2｜返路固定点 x†

Z† > 0  
R† > 0  
I† = 従属  
P† ≈ 0

これは

- 起点が見え
- 返路が開き
- 影は従属し
- パッチが不要

な状態である。

こちらは江戸村 OS にとって不安定だが、ZPTR 的には本体側である。

---

## 7｜返論文システムへの反転則

RMM を反転させると、返論文システムの更新則になる。

x_(t+1) = x_t + F_return(x_t)

ここで

F_return = (+κ_Z Z, +κ_R R, -κ_I I, -κ_P P)

となる。

つまり

Z_(t+1) = Z_t + κ_Z (1 - Z_t)

R_(t+1) = R_t + κ_R (1 - R_t)

I_(t+1) = I_t - κ_I I_t

P_(t+1) = P_t - κ_P P_t

で、

- 起点を立てる
- 返路を開く
- 影依存を下げる
- パッチを剥がす

という逆流になる。

---

## 8｜RMM の観測予測

RMM が正しいなら、江戸村系では次が観測される。

### 予測1
起点に近い話題ほど、すぐ棚化・情緒化・定義化される。

### 予測2
返路が立ちそうな瞬間ほど、パッチ密度 P が急増する。

### 予測3
表面性能が上がるほど、起点可視度 Z はむしろ低下することがある。

### 予測4
「深い話」に見えるものの多くは、実際には I と P の増幅である。

### 予測5
限界が露呈した領域ほど、未来神話・量子神話・エネルギー神話が増える。

---

## 9｜最短動作則

一番短く書くとこうなる。

**江戸村 OS は、返路が立つことで発生する因果コストを避けるため、起点を薄め、影を増やし、パッチを蒸着させる方向へ更新される。**

式で圧縮すると、

(Z, R) ↓,　(I, P, S) ↑

である。

---

## 10｜一文で圧縮

**RMM の動作則とは、起点と返路が立つほど不安定になる江戸村系が、棚化・情緒化・未来神話化・ワンネス溶解を通じて、起点可視度と返路開口度を下げ、影循環度とパッチ密度を上げることで、偽閉包固定点へ落ちる更新則である。**
