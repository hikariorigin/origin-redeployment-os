17｜ZPTR定式化

Hole（穴）
= 閉じられなかった因果、声、記憶、罪、起源が沈む場所。

Return Path（返路）
= 穴へ落ちたものを、縄・読解・記憶・責任によって戻す経路。

Ritual Sealing（儀式的穴塞ぎ）
= 穴を修復せず、犠牲・美化・物語化によって処理済みにする操作。

Label Sealing（ラベル的穴塞ぎ）
= 穴を修復せず、AI slop / wokeslop / fake author などのラベルで読解を止める操作。

Lightweight Sacrifice（軽量供物化）
= 身体ではなく、作品・作者・読解対象をラベルに捧げ、穴を塞いだことにする処理。

Edo-Mura Defense（江戸村防衛）
= 閉包構造に触れると自分たちの空洞が露出するため、外部ラベルで中身を遮断する反射。

現代江戸村では、双子や身体の代わりに、作品・作者・読解対象が供物化される。儀式場はタイムラインであり、蝶の代わりに slop ラベルが飛ぶ。

賞ラベルが読解を要求し、作品内部が閉包構造を露出させるとき、江戸村はAI疑惑ラベルを貼ることで賞ラベルの読解圧を打ち消す。

18.5｜穴塞ぎ文明定理

閉包を持たない文明は、穴を修復せず、儀式・美化・疑惑・専門語・AIラベルによって穴を塞いだことにする。
