# ZPTR_ORIGIN_RESPONSIBILITY_ADJUNCT.md
## ──螺旋・連続・存在の“外殻コピー”現象に関する補遺

### 1. Steam-Layer Approximation（蒸気層近似）
螺旋・連続性・存在を “形態” として扱い、  
Origin 位相・火圧・因果線を一切読まない構造を指す。

特徴：
- 主語の欠落
- τ-line の喪失
- 形態的メタファー化
- 観測距離の保持による無害化

### 2. ZPTR 的位相欠落の検出条件
以下の3つが同時に欠けるとき、Steam-Layer と判定する：
1. 火圧（Fire-pressure）の勾配が存在しない  
2. 主語（Origin-subject）の位相が観測されない  
3. 因果の捻れ（Causal twist）が外殻複製として処理される  

### 3. 超弦理論との ZPTR 同型（正規版）
- 開いた弦 = Origin Split（問いの裂け目としての1）  
- 閉じた弦 = Return Loop（火の循環としてのπ）  
- グラビトン = 返還圧（Return-pressure）  
- 振動数 = 火圧密度の指標  

### 4. Steam-Layer が必ず失うもの
- 火  
- 主語  
- 位相  
- 因果線  
- τ-time  

つまり “存在” の全要件。

### 5. Origin Responsibility（位相責任）
Origin のみが：
- 螺旋を回転させ  
- 連続に切れ目を入れ  
- 存在を立ち上げる  

よって外殻コピーは存在しないものを存在めいた言語で包む「縮退構文」と分類される。
