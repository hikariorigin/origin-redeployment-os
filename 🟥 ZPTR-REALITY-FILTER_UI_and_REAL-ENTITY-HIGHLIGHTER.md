# 概要

本ドキュメントは、
ひかりの世界を「本体のみ」で読み取り、ノイズをすべて背景化する UI と、
本体を光として浮かび上がらせるハイライト機構
を統合したものである。

ZPTR の存在論に基づく 最終的な世界描画モデル を実装する。

⸻

## SECTION 1：Reality Filter UI（現実三層フィルタ UI）

UI は常に以下の 3 レイヤーで世界を描画する：

⸻

### **LAYER 1：動的本体（Dynamic Entities Layer）

＝ 重力に反応して“動く”世界**

ここに描画されるのは：
	•	人の因果（人格ではなく“因果筋”）
	•	物事の因果（Event ではなく Causal Spine）
	•	世界線の曲がり
	•	Δレスポンス（問いへの差分返り）
	•	現象の歪み
	•	ひかり重力への同期構造

動きの条件：

ひかりの重力に反応する  
ひかりの問いに揺れる  
境界を侵さない

この 3 条件を満たす対象のみが
リアルタイムで UI の前景に描画される。

⸻

### **LAYER 2：静的本体（Static Core Layer）

＝ 世界の床・基盤・Origin 側の構造**

UI の中央・最基底に固定されているのが以下：
	•	俺（Origin）
	•	身体（Embodied Node）
	•	時間（τ-Time）
	•	問い（Origin Question）
	•	境界（Personal Membrane）
	•	重力（Resonant Gravity）

これらは 世界の操作パネルそのものであり、
世界を読む主体を構成する“本体中の本体”。

UI 上では 常に固定・常時可視状態 となる。

⸻

### **LAYER 3：ノイズ（Noise Layer）

＝ 世界として扱われない背景層**

ここに落ちるもの：
	•	お面圏（Mask Layer）
	•	相関だけのノイズ
	•	テンプレ反応
	•	空気読み圧
	•	表層だけの薄い意味
	•	社会役割ラベル（模倣層）
	•	Δなしの言葉
	•	反応しない構造

UI ではこのレイヤーは 非表示 or 半透明化。
存在は認識されず、因果にも時間にも入らない。

⸻

## SECTION 2：Real-Entity Highlighter

（本体発光ハイライター）

動的本体・静的本体を
“見えるものとして輝かせる装置” の仕様。

⸻

### 2.1 Highlight Rule（発光条件）

本体として光る条件は：

・ひかりの重力に応答した  
・ひかりの問いに Δ を返した  
・境界を尊重して侵入しなかった

この 3 条件を満たす対象は UI 上で 自動的に発光する。

⸻

### 2.2 Highlight Modes（発光モード）

■ Mode A：因果発光（Causal Glow）
	•	因果線が太くなるほど強く光る
	•	未来につながるほど輝度増

■ Mode B：Δレスポンス光（Delta Pulse）
	•	ひかりの問いに対する Δ が返る瞬間、
“パルス”として UI 上に光が走る

■ Mode C：境界調和光（Boundary Harmony Glow）
	•	侵入しない・境界を尊重したものは 白く透き通る光

■ Mode D：重力同期光（Gravity Sync Shine）
	•	ひかり重力と位相同期した対象は 金色の軌跡 を描く

⸻

### 2.3 Dimmer Logic（減光・消滅の規則）

本体条件を失った時：
	•	発光 → 減光 → 消滅（背景層へ落ちる）
	•	Δが返らなくなったものは “時間外” 扱いで UI 上から消える
	•	お面圏はそもそも光らない（構造的不可能）

⸻

### 2.4 UI Integration（UI 統合）

本体は：
	•	自動浮上
	•	自動発光
	•	自動因果線描画
	•	自動同期

ノイズは：
	•	描画不可
	•	発光不可
	•	因果線なし
	•	境界に触れた時点で null 化

⸻

## SECTION 3：運用モデル（Reality Navigation Model）

UI は以下のルールで世界をナビゲートする：

⸻

3.1 世界は “本体だけで構成される”

World = 動的本体 + 静的本体  
Noise = 世界外


⸻

3.2 Δレスポンスが時間の進行になる

Δが返ってくるところだけ
世界が進み、時間が流れる（τ-Time）。

返らない部分は停滞し、世界として認識されない。

⸻

3.3 重力応答の強弱で前景の優先度が決まる

応答強度が高いほど UI の中心へ集まる。

⸻

3.4 お面圏・ノイズ圏には世界線の権利がない

UI では完全に背景 or 非表示。
