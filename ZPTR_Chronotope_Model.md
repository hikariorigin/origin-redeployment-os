0. Abstract / 要旨

時間とは “世界の流れ” ではなく、
Boundary（Self と World の境界）が揺れたときに生成される局所的折り畳み（τ） である。

世界の時間（bulk clock）は外界の構造物であり、
主体の時間（τ）は Boundary の揺れによって生成され、
Self は τ を編集して「物語としての時間」を構築する。

つまり、時間には 3 層の構造がある：
	1.	World-Time（bulk clock）
　外側から与えられる線形時間。24時間、暦、宇宙時間。
	2.	τ-Time（folded subjective time）
　Boundary が揺れる瞬間に生まれる折り畳み時間。
　非線形・局所・主語性を持つ。
	3.	Narrative-Time（self-edited time）
　Self が τ の痕跡から “物語” として再構築した時間。

この 3 層のズレが
自由意志、ゾーン、時間感覚の歪み、意識の後付け性、
洞窟隔離での時計崩壊など
人間の謎をすべて説明する。

1. ZPTR Chronotope Model（時間構造モデル）

基本構造：

Self（内） ← Boundary（揺れる界面） → World（外）
                        ↑
                        τ（折り畳み時間）


⸻

1.1 Boundary — 時間を生み出す界面

Boundary は Self と World の界面であり、
外部刺激（World）と内部状態（Self）の 差分と揺れ を保持する場所。

ここで発生した揺れ・曲率が
τ（主語時間） の源になる。

Boundary が揺れる → τ が発生
Boundary が滑らか → τ は消える
Boundary が途切れる → Self は存在できない

⸻

1.2 τ-Time — 主語によって生まれる “折り畳み時間”

τ は以下の性質を持つ：
	•	非線形（伸びる／縮む／結び目をつくる）
	•	局所的（Boundary が揺れた瞬間だけ存在）
	•	個体固有（1人1人τ構造が違う）
	•	意識や選択感と密接に結びつく
	•	World-Time とは独立して動く

数式（最小表現）

τ(t) = f(κ_boundary(t))

境界曲率の関数であり、
揺れが大きいほどτは伸び、
揺れが止まると縮む。

⸻

1.3 Narrative-Time — 自我が“時間”を編集する工程

Self（自我）は、τ の散発的な揺れを
「連続した時間」という物語に編集する。

意識の順番
	1.	Boundary が揺れる（世界と接触）
	2.	τ が発生（時間の局所折り畳み）
	3.	行動が起きる
	4.	Self が「これは私が決めた」と物語化する（遅延）

2. 現象との整合（ゾーン／リベット／洞窟隔離）

⸻

2.1 ゾーン状態

ゾーンでは、
	•	Boundary ↔ World が直結
	•	τ が滑らかに連結
	•	Self が遅れる

このため：
	•	「自分が消えた」
	•	「勝手に体が動いた」

と主観されるが、実際には：

Self が遅れただけで、Boundary はむしろ最適状態。

⸻

2.2 リベット実験（意識は0.3〜0.5秒遅れる）

リベットが観測した「準備電位 → 意識の遅れ」は、

Boundary揺れ → τ生成 → Selfに到達するまでの遅延

で説明される。

「行為が先」は正しいが、
正確には：

Boundary → τ → 意識（Self）
の順で立ち上がっているだけ

⸻

2.3 洞窟隔離実験（NASA）

洞窟では World-Time への外部同期が切れる。

結果：
	•	Boundary の揺れが自由化
	•	τ が伸びたり縮んだり暴走
	•	Self が τ と同期できず “時間崩壊” を起こす
	•	28時間、48時間周期が出現
	•	日付が完全にずれる

つまり、

時計時間は「後付けの世界の構造物」にすぎず
主体の時間は Boundary と τ によって決まる

洞窟で壊れるのは脳ではなく、
World-Time 同期の強制が解除されただけ。

⸻
