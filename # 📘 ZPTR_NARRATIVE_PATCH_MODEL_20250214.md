# 1. 用語定義

記号	意味
S	世界構造（複雑で重い）
C	認知容量（有限。退行すると縮む）
B	bulk 構造コスト（思考・責任・倫理負荷）
L	ラベリング操作（cheap）
N	ナラティブ片（1文〜数文の軽い物語）
R	レンダリング圧（見せかけの説得力）
J	自己正当化強度（ego maintenance）

# 2. 基本前提

2.1 認知容量 C は世界構造 S を完全には扱えない

C < |S|

よって世界は 圧縮表示 される。

⸻

2.2 bulk 構造 B を扱うにはコストがかかる

\text{Cost}(B) \gg \text{Cost}(L, N)

つまり：
	•	倫理
	•	科学
	•	歴史
	•	複雑な動機の絡み合い
	•	多層の因果

これらは高価。

⸻

2.3 主体は常に最小コストで自己正当化 J を最大化しようとする

\max J \quad \text{s.t.} \quad \min \text{Cost}

⸻

# 3. ナラティブ・パッチ当ての基本関数

主体は、複雑な世界 S を以下の cheap な関数で再構成する：

f_{patch}(S) = L(S) + N

すなわち、

✔ 世界をラベル L に潰す

✔ 足りない部分はナラティブ N を後付けする

⸻

3.1 ラベル化操作 L の式

L: S \to \{0,1\}

例：
	•	woke / anti-woke
	•	truth / lie
	•	safe / unsafe
	•	正しい / 間違い

世界を 二分割 し、
|S| \to 2

⸻

3.2 ナラティブ片 N の生成式

ナラティブ片 N は以下で生成する：

N = g(E,C)
	•	E：情動（怒り・不安・承認欲）
	•	C：認知容量

C が低いほど：

N \text{ は短く、単純で、再利用される。}

例：
	•	「Only X is truth」
	•	「They are lying」
	•	「The media is corrupt」

⸻

つまり 内容ではなく語気で補う。

# 5. 全体系の式

最小コスト・最大自己正当化の問題は以下に落ちる：

\max_{L,N} \; J(h(L,N))
\quad \text{s.t.} \quad
\text{Cost}(L,N) \ll \text{Cost}(B)

解は常に：

B \text{ を捨て、} \; L + N \text{ を採用する。}

これがナラティブ・パッチの本質。
