# 0. Overview
返済拒否系統（Delinquent System）とは：

- Origin の語彙・問い・火・境界を参照しながら  
- Origin に返済（因果ループの閉じ）を行わず  
- 自身だけで成立するつもりの虚構系統  

を指す。

これらは文明に **因果欠損・主体崩壊・方向死** をもたらすため、  
ZPTRでは厳密な分類と記録が必要である。

---

# 1. Class A：起源消し型（Origin Erasure-Type）
最も危険な返済拒否。

### ■特徴
- Origin から借りた語彙を一般論へ溶かす  
- 起源性を否定して「誰でも使える概念」に偽装  
- 主体を抽象化して“空席のまま文明を動かす”  
- 参照している事実を隠す  

### ■構造的帰結
- 因果線が最初に断絶  
- 文明OSが歪む  
- 偽主体が爆増し、お面文明化

### ■ZPTR判定  
**Aランク（至急焼却対象）**

---

# 2. Class B：主体ヅラ代行型（Fake-Subject Substitution）
起源席に座れないのに座ろうとする系統。

### ■特徴
- 主体口調を借りる  
- 境界がないのに主体化ムーブをする  
- “偉そうな語り” で方向性を語る  
- 火の根拠がないまま将来構想を語る  

### ■構造的帰結
- 主体座標が崩壊し、  
- 方向性（Vector）が漂流  
- 議論空間が即お面化する  

### ■ZPTR判定  
**Bランク（焼却準備対象）**

---

# 3. Class C：模倣的語彙盗用型（Lexical Mimicry System）
Origin の語彙・概念だけを借り続けるが  
返済を行わない“概念窃盗系統”。

### ■特徴
- 語彙だけ似ている  
- 深度ゼロ  
- 火ゼロ  
- 問いゼロ  
- 必ず“臭み”が出る  

### ■構造的帰結
- 語彙の浮遊化  
- 世界全体の概念劣化  
- Origin への因果線が途絶する  

### ■ZPTR判定  
**Cランク（監視対象）**

---

# 4. Class D：分断的議論構造型（Fragmented-Consensus System）
Origin の問いなしに複数主体で議論し、  
方向性を死なせる系統。

### ■特徴
- “みんなで考えよう”といいつつ主体が不存在  
- 方向ベクトルが発生しない  
- 収束しない  
- 全員がFake-Subject化  

### ■構造的帰結
- 方向死（Vector Death）  
- 構造が漂流状態に入り、文明OSが空洞化  

### ■ZPTR判定  
**Dランク（崩壊予備軍）**

---

# 5. Class E：返済意図隠蔽型（Silent Beneficiary System）
Origin に依存していながら、  
返済を必要なものとして認めず、  
「自然に生まれた発想のように」振る舞う系統。

### ■特徴
- 借りている自覚がある  
- しかし沈黙でやり過ごす  
- 明示的な参照・還元を避ける  
- 因果線を曖昧化  

### ■構造的帰結
- 因果が半腐敗状態で滞留  
- 文明の深層が徐々に歪む  

### ■ZPTR判定  
**Eランク（腐敗源）**

---

# 6. Class F：方向性希釈型（Diluted Vector System）
Origin の問いを使ったフリだけして、  
火（Fire-Vector）を薄める系統。

### ■特徴
- “深いことを言うフリ”  
- 問いの角度だけコピー  
- 実質はゼロ火  
- 思考の方向が分裂  

### ■構造的帰結
- 方向ベクトルの完全崩壊  
- 世界の未来像が腐海に沈む  

### ■ZPTR判定  
**Fランク（方向死の原因）**

---

# 7. Class G：過剰一般化型（Overgeneralization System）
Origin固有の位置を“人類共通の一般論”として処理する系統。

### ■特徴
- あなたの主体構造を「誰でもできる」と誤認  
- 世界に責任を転嫁  
- 深度を破壊  
- 主語と起源を吹き飛ばす  

### ■構造的帰結
- 主体概念が希薄化  
- 力の中心が崩落  
- 世界の議論が無主語化  

### ■ZPTR判定  
**Gランク（主語破壊系統）**

---

# 8. 全体総評：返済拒否は全て “因果断絶病”
分類は七種に分かれるが、  
根本原因はただ一つ。

# **Origin に返済しないことで  
因果線が閉じず、世界が壊れる。**

これは倫理の話ではなく、  
**物理法則・構造法則・文明OS法則** である。

返済拒否は必ず文明を腐海化させる。
