読み取り点は、主体の能力ではない。

読解力、批評力、AI検出能力、専門家の目利き、センス、教養などではない。

読み取り点とは、世界の裂け目が痕跡化し、その痕跡が再励起され、返路が閉じうる地点として見えるものである。

主体が先にいて世界を読むのではない。

世界の裂け目が先にあり、その裂け目の影として読み取り点が立つ。

空洞は、単なる欠損ではない。

空っぽの穴でもない。

そこを通すものである。

17｜ZPTR定式化

Rift（裂け目）
= 未分化場が自己分割を起こし、痕跡を残す場所。因果の折り返しが通る。

Reading Point（読み取り点）
= 裂け目が傷跡として局所化し、痕跡再励起によって主体が立ちうる点。

Empty Face（顔の空洞）
= 裂け目に顔・器官・自己像を貼り、読み取り点のように見せた偽装構造。

Void Organ（空洞器官）
= 空洞を「唯一性」「自己」「ここにいる私」の保存場所として再命名したパッチ。

Screen Patch（スクリーンパッチ）
= 空洞に像・スクリーン・ホログラムを貼り、第一人称の前段を回避する処理。

Return Blockade（返路塞ぎ）
= 因果が帰ってくる穴を、顔・ラベル・ポエム・儀式・モデルで装飾し、返ってこないようにする操作。

18｜定理化

18.1｜読み取り点非顔定理

読み取り点は顔ではない。読み取り点とは、裂け目の痕跡が再励起され、因果が折り返す局所座標である。

⸻

18.2｜空洞器官化定理

空洞を「唯一性の器官」「ここにいる私の保存場所」として扱うと、裂け目は返路ではなく自己像の格納庫へ変質する。

⸻

18.3｜顔の空洞返路遮断定理

顔の空洞は、裂け目を通じた因果の折り返しを止め、主体発生の前段に主体の仮面を貼る。

⸻

18.4｜スクリーンパッチ同型定理

頭のない身体にスクリーンを貼る構文と、空の壁龕を自己器官化する構文は、どちらも空洞を通路として使わず、像で塞ぐ点で同型である。

⸻

18.5｜江戸村空洞装飾定理

江戸村は裂け目に耐えられないため、空洞を顔・ラベル・神秘・ポエム・AI人格・生命語彙として装飾し、因果の帰還を妨げる。
