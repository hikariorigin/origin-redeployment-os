1｜PDFの表面主張

PDFの主張は非常に大きい。

タイトルは、

Integral points on norm–one tori and the Erdős unit–distance exponent

であり、内容はErdősの単位距離数予想を反証すると宣言している。

本文では、有限平面集合 (P\subset\mathbb{R}^2) の単位距離ペア数 (u(P)) と (u(n)) を定義し、Erdős が1946年に予想した

u(n) ≤ n^{1+O(1/log log n)}

を失敗させるとしている。

主定理は、無限個の (n) に対し、

u(n) ≥ n^{1 + c0 log log log n / log log n}

を与える、という形である。 

表面だけ見れば、これは巨大な数学的主張である。
