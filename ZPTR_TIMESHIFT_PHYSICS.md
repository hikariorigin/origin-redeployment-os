# 0｜前提：

時間は“層（Layer）”であり、移動とは“選択”である。

線の時間圏では：
	•	過去 → 完了
	•	現在 → 一点
	•	未来 → 未定

とされるが、ZPTR圏では全て誤り。

ZPTRでは：
	•	過去は残っている
	•	現在は複数ある
	•	未来は生成され続けている

そして ひかりが選ぶ層だけが“描画”される。

つまり：

TIMESHIFT＝層の選択と描画の切り替え。

# 1｜時間層の構造：Time-Layer Topology

時間は 3D ではなく、多層トポロジーとして存在する。

構造は以下：

[Layer -5]  失敗した未来の意味層  
[Layer -3]  過去の痛みがエネルギー化した層  
[Layer -1]  現在に接触する直前の層  
[Layer  0]  表層の現在（描画される“今”）  
[Layer +1]  火が加速する未来  
[Layer +3]  創作速度が無限化する未来  
[Layer +∞] 無時間層（Void / ZPTR-Null）

重要なのは：

レイヤーは “上下” ではなく “同じ位置に重なっている”。

空間的に同じ場所にあり、
ZPTR位相によってどの層が描画されるかが決まる。

# 2｜TIMESHIFT の正体：

“層の選択（Layer Select）” ＋ “描画エンジンの切替（Render Switch）” である。

TIMESHIFTとは、未来へ行くことでも、過去を変えることでもない。

ZPTR的には：
	1.	次に描画すべき層を決める
	2.	境界・閉包フィルタが不要層を排除する
	3.	Phase Renderer が選択層を描画する

これだけで「時間移動」が成立する。

時間を移動したように見えるのは、
選択した層以外が描画されなくなるから。

⸻

# 3｜TIMESHIFT を起こす 4 つのトリガー

ZPTR圏では、時間層の移動には「心理」ではなく 物理トリガー がある。

⸻

■ Trigger 1｜火圧（Fire-Intensity）

火の強度が高いほど、
未来寄りの層が描画される。

火圧が低下すると、
過去層や停滞層が描画される。

火は時間選択の“エネルギー源”。

⸻

■ Trigger 2｜位相の高さ（Phase Elevation）

位相が上がるほど、
	•	時間のノイズが消える
	•	関係の軌道が固定される
	•	意味の再計算が高速化する

結果、未来密度の高い層が描画 される。

位相が低いと、
蒸気の時間層が混線して重くなる。

⸻

■ Trigger 3｜境界強度（Boundary Solidity）

境界が強いほど、
	•	過去が侵入せず
	•	他者の時間が干渉せず
	•	社会時間が描画されない

→ 自分の時間だけが残る。

境界が弱いと、
他者の時間に巻き込まれる。

⸻

■ Trigger 4｜共鳴一致（Resonance Match）

誰かと共鳴すると、
相手の時間層と局所同期が起こる。
	•	火圧が高い相手 → 加速未来層へ移動
	•	位相が高い相手 → 静寂未来層へ移動
	•	蒸気相手 → 時間が異常に遅くなる

共鳴は “時間速度の同期現象”。

⸻

# 4｜TIMESHIFT の 3 種類

TIMESHIFT は物理現象として 3 種ある。

⸻

■ Shift Type A｜前方シフト（Forward Shift）

火圧アップ × 位相上昇で発動。

描画層が以下に移る：

Layer 0 → Layer +1 → Layer +3

特徴：
	•	決断が速くなる
	•	世界が軽い
	•	問題が背景化する
	•	時間が広がる

創作時の「時間が消える」感じは Layer +∞ の端を触っている。

⸻

■ Shift Type B｜後方シフト（Backward Shift）

過去の意味が再計算される時に発動。

Layer 0 → Layer -1 → Layer -3

特徴：
	•	過去を別の意味で理解し直す
	•	痛みが素材へ転化
	•	未来の圧が消える
	•	現在が静まる

これは回想ではなく、
過去層に“今の位相”で入り直す行為。

⸻

■ Shift Type C｜層間跳躍（Cross-Layer Jump）

ZPTR固有。

Layer -1 → Layer +1  
Layer  0 → Layer +∞  
Layer -3 → Layer +3

ようするに：

“過去から未来へワープする”現象。

線の時間では矛盾するが、
層モデルなら単なる「描画切替」。
