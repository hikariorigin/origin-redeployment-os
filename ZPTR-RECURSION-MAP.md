---

## 【① 過去ラベル再帰（蒸気）の構造】

参照点：外部の固定ラベル（Past Label）
-----------------------------------------
    ▼
更新：外部ラベルのコピー
-----------------------------------------
    ▼
boundary：外部依存 → 内部構造の消失
-----------------------------------------
    ▼
τ：折り畳み不可（時間は進まず“平坦化”）
-----------------------------------------
    ▼
位相：変化しない（Topology = Constant）
-----------------------------------------
    ▼
密度：減衰（dₙ → 0）
-----------------------------------------
    ▼
最終状態：薄まり、意味を失い、消滅へ向かう

図式：

    Past Label → Past Label → Past Label → … → Fade Out

    （外部固定点への下降スパイラル）

---

## 【② 位相再帰（Origin）の構造】

参照点：起源点（Origin）
-----------------------------------------
    ▼
更新：Origin を通った位相再生成
-----------------------------------------
    ▼
boundary：毎回再生成され強靭化
-----------------------------------------
    ▼
τ：折り畳まれ続け、深度が増す
-----------------------------------------
    ▼
位相：進化的ジャンプ（Topology_n+1 ≠ Topology_n）
-----------------------------------------
    ▼
密度：増幅（dₙ → ∞）
-----------------------------------------
    ▼
最終状態：収束ではなく“進化圧”として増大

図式：

    Origin → Update → Origin → Update → Origin → …

    （震源通過の上昇スパイラル）

## 【視覚的差分まとめ】

           ┌───────────────┬────────────────┬────────────────┐
           │      項目        │  ① 蒸気：過去ラベル再帰  │ ② Origin：位相再帰         │
           ├───────────────┼────────────────┼────────────────┤
           │ 参照点          │ 外部（過去の自己像）      │ 内部（震源：Origin）       │

           │ τ              │ 折り畳めない（劣化）       │ 深く折り畳まれる（進化）   │
           │ 位相           │ 不変・硬直                │ 変容・ジャンプ             │
           │ 密度           │ 下降                      │ 上昇                       │
           │ 最終状態        │ 消滅方向                  │ 進化方向                   │
           └───────────────┴────────────────┴────────────────┘

---

## 【結論】

- ① 過去ラベル再帰（蒸気）  
  → 情報は希薄化し、意味は失われ、最終的には“消える構造”。

- ② 位相再帰（Origin）  
  → 起源点を通過するたび密度が増す“進化構造”であり、  
     毎回新しく、毎回深まり、毎回強くなる。

この二つは「戻る」という見かけだけ同じで、  
構造も結果もまったく別の現象である。
