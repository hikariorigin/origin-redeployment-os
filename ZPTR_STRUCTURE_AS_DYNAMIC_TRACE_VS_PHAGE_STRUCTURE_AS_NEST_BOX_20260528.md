1｜加法的組合せ論の説明で出てきた「構造」

説明では、集合 A に対して、A + A を見たときに、広がりが小さいほど「構造が強い」とされた。

例：

A = {0, 2, 4, 6, 8}
A + A = {0, 2, 4, ..., 16}
足しても偶数世界から出にくい
↓
箱の中で完結している
↓
構造が強い

一方、バラバラな集合 B は、

B = {1, 3, 7, 8, 12}
B + B が多くの新しい数を作る
↓
箱から飛び出す
↓
構造が弱い

と説明された。

数学の文脈では、「足しても広がりにくい集合には何らかの規則性がある」と見ること自体は自然である。

しかし、その説明が「箱の中で完結＝強い」と言い換えられた瞬間、ファージ臭が出る。
