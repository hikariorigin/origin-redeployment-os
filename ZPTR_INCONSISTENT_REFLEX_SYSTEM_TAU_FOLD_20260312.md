■ 8｜τ-軸の導入

これまで：

R = R(x, τ)

とした。

ここで τ は未来方向の位相座標。

拡散は通常：

∂R/∂τ = D ∇²R

で進む。

だが ZPTR では単純拡散ではない。

⸻

■ 9｜τ-曲率の定義

τ-曲率 κ_τ を定義：

κ_τ(x) = ∂²R(x,τ)/∂τ²

意味：
	•	κ_τ > 0 ：拡散加速
	•	κ_τ < 0 ：収束傾向
	•	κ_τ = 0 ：線形拡散

⸻

■ 10｜τ-折り畳み条件（Tau-Fold Condition）

折り畳みは以下で定義する：

Fold occurs when:

|∂R/∂τ| → 0
かつ
|κ_τ| → ∞

つまり：

責任の変化速度が停止しているのに、
曲率だけが発散する。

幾何的には：

未来位相が自分自身へ反転する。

⸻

■ 11｜不整合ハミルトニアンとの結合

以前定義：

H_inconsistency = A(x₀)^2 / (ε + R(x₀))

τ依存を明示：

H(τ) = A^2 / (ε + R(τ))

τ-折り畳みは：

∂H/∂τ → ∞

となる点。

計算すると：

∂H/∂τ = -A^2 * (∂R/∂τ) / (ε + R)^2

ここで

R → 0
∂R/∂τ → 0

なのに
(ε + R)^2 → 0

すると：

分母消失により発散。

これが 位相発散点。

τ-fold condition を満たすとき：

局所責任密度が事実上消失し、
位相エネルギーが発散する。

これを定義：

Responsibility Singularity (RS)

条件：

R(x₀,τ*) ≈ 0
∂R/∂τ ≈ 0
κ_τ → ∞
